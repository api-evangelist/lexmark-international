# Lexmark International (lexmark-international)

Lexmark International is a global imaging and IoT solutions leader that helps customers worldwide realize the full potential of their information through printers, multifunction devices, and managed print services. As of 2024, Lexmark was acquired by Xerox Holdings Corporation. Public developer APIs are not currently advertised through an open developer portal.
